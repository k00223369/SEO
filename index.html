Higher Diploma Creative Multimedia Programming B.Sc

Level 8

This Diploma Creative Multimedia Programming Level 8 Higher Diploma is a one-year ICT skills conversion course. The course is designed to give students the essential skills to gain employment. The aim of the course is to give students the skills to make them more employable today. The course is cutting-edge when it comes to technology. The year is broken up into three semesters. The pace of delivery and intensity of the programme demand strong commitment and dedication. Learners will engage in active learning through. 1) Attendance on-site at LIT. 2) Lectures. 3) Tutorials. 4) Practicals. 5) Lab-based lectures. 6) Seminars. 7) Workshops

Semester 1 Modules

Web Programming

We start off learning coding languages such as Javascript.

Web Development

Build a foundation of HTML and CSS code in this module.

Interactive Multimedia Design

Get started with the Adobe Creative Suite. Creating graphics.

Multimedia Digital Video

This is where you learn teamwork. Create videos with your classmates. You learn camera, directing and editing skills.

3D Graphics

Learn 3DS Max and modelling in a 3D environment, as well as animation.

Multimedia Database Systems and Design

Learn coding language SQL. This is where you learn the back-end of software development.

Semester 2 Modules

Advanced programming

Advanced programming builds on the foundation of Semester 1 programming. Learn all about PHP, Codeigniter and XML.

Evolving Web Technology

We develop the skills for responsive web design. We also discover the skills of search engine optimisation and how to use Google Analytics to get more from your website.

Interactive Interface Design

Build your online portfolio in this module. Also gain knowledge on user experience, uses cases and needfinding in regard to website/app design.

Multimedia Project

This is the final year project. Choose a topic you find interesting and design a website.

Human Cognition and Communication

Here we learn about human interaction with websites. We gain a deeper understanding of user experience in this module.

Semester 3 - Work Placement

Undertake a 3 month work placement to gain experience in the wonderful world of Creative Multimedia Programming.
